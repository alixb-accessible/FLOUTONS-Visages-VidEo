# FLOUTONS-Visages-VidEo
Floutage des visages sur les vidéo.

Un guide utilisateur est intégré.

Mode d'emploi dépliable avec instructions PC et mobile
Détection mobile avec avertissement automatique
PC - Détection auto avec BlazeFace
Édition manuelle : ajouter/déplacer/redimensionner/supprimer des ovales
Tout ajustable : taille des ovales, intensité du flou (5-50px)
Contrôles clavier : Suppr pour effacer, Tab pour naviguer
Barre d'accessibilité Faites Mieux intégrée
Responsive : s'adapte mobile/tablette/PC
Confidentialité : 100% local

Fonctionnement :

Cliquer + glisser sur la vidéo pour créer un ovale
Glisser le centre pour déplacer
Point blanc en bas à droite pour redimensionner
Bouton ✕ rouge ou touche Suppr pour effacer
Sur mobile : pas de détection auto (trop lourd), mode manuel seulement.
